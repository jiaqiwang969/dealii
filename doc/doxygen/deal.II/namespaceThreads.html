class  ThreadLocalStorage
 A class that provides a separate storage location on each thread that accesses the object. This class offers ways so that every thread that accesses it has its own copy of an object of type T. In essence, accessing this object can never result in race conditions in multithreaded programs since no other thread than the current one can ever access it. More...
