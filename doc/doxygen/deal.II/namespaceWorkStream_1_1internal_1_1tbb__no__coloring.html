Detailed Description

A namespace for the implementation of details of the WorkStream pattern and function. This namespace holds classes that deal with the second implementation described in the paper by Turcksin, Kronbichler and Bangerth (see workstream_paper ). Here, no coloring is provided, so copying is done sequentially using a TBB filter. Even though this implementation is slower than the third implementation discussed in that paper, we need to keep it around for two reasons: (i) a user may not give us a graph coloring, (ii) we want to use this implementation for colors that are just too small.
