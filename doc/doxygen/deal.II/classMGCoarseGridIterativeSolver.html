Detailed Description

template<class VectorType, class SolverType, class MatrixType, class PreconditionerType>
class MGCoarseGridIterativeSolver< VectorType, SolverType, MatrixType, PreconditionerType >

Coarse grid multigrid operator for an iterative solver. This class provides a wrapper for a deal.II iterative solver with a given matrix and preconditioner as a coarse grid operator.
