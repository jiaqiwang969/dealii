797 * <a name="codeDiffusionexplicit_methodcode"></a> <h4><code>Diffusion::explicit_method</code></h4>
798 * This function is the driver for all the explicit methods. At the top it initializes the time stepping and the solution (by setting it to zero and then ensuring that boundary value and hanging node constraints are respected; of course, with the mesh we use here, hanging node constraints are not in fact an issue). It then calls <code>evolve_one_time_step</code> which performs one time step. Time is stored and incremented through a DiscreteTime object.
799 * For explicit methods, <code>evolve_one_time_step</code> needs to evaluate @f$M^{-1}(f(t,y))@f$ , i.e, it needs <code>evaluate_diffusion</code> . Because <code>evaluate_diffusion</code> is a member function, it needs to be bound to <code>this</code> . After each evolution step, we again apply the correct boundary values and hanging node constraints.
