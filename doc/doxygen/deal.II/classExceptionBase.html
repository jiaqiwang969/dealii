Detailed Description

This class is the base class for all exception classes. Do not use its methods and variables directly since the interface and mechanism may be subject to change. Rather create new exception classes using the DeclException macro family. See the Exceptions and assertions This module contains classes module for more details on this class and what can be done with classes derived from it.
