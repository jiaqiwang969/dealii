Detailed Description

template<int dim, int spacedim>
struct internal::DataOutImplementation::ParallelDataBase< dim, spacedim >

A data structure that holds all data needed in one thread when building patches in parallel. These data structures are created globally rather than on each cell to avoid allocation of memory in the threads. This is a base class for the AdditionalData kind of data structure discussed in the documentation of the WorkStream class. The cell_to_patch_index_map is an array that stores for index [i][j] the number of the patch that associated with the cell with index j on level i. This information is set up prior to generation of the patches, and is needed to generate neighborship information. This structure is used by several of the DataOut* classes, which derived their own ParallelData classes from it for additional fields.
