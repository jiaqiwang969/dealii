Detailed Description

This class is much like the Selection class, but it allows the input to be a comma-separated list of values which each have to be given in the constructor argument. The input is allowed to be empty or contain values more than once and have an arbitrary number of spaces around commas. Of course commas are not allowed inside the values given to the constructor. For example, if the string to the constructor was "ucd|gmv|eps", then the following would be legal inputs: "eps", "gmv, eps", or "".
