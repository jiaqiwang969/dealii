Detailed Description

template<int dim>
class PolynomialsRannacherTurek< dim >

Basis for polynomial space on the unit square used for lowest order Rannacher Turek element. The i-th basis function is the dual basis element corresponding to the dof which evaluates the function's mean value across the i-th face. The numbering can be found in GeometryInfo.
