*Base class for solver classes using the PETSc solvers Since solvers in PETSc are selected based on flags passed to a generic solver object basically all the actual solver calls happen in this class and derived classes simply set the right flags to select one solver or another or to set certain parameters for individual solvers Optionally the user can create a solver derived from the SolverBase class and can set the default arguments necessary to solve the linear system of equations with SolverControl These default options can be overridden by specifying command line arguments of the form *[2.x.0] *sp_ *For **sp_monitor_true_residual prints out true residual norm (unpreconditioned) at each iteration and[2.x.2] **- sp_view provides information about the linear solver and the preconditioner used in the current context.
